# OTP Login System

A secure OTP-based login system using React, Vite, and SendGrid.

## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and add your SendGrid API key
4. Run locally: `npm run dev`

## Deploy to Vercel

1. Push to GitHub
2. Import project in Vercel
3. Add environment variables in Vercel dashboard
4. Deploy!
